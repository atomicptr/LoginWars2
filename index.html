Do you use multiple accounts? Tired of retyping your credentials? Login Wars 2 is a fix for this problem! Open the game with any account in just a single click!

No more need to save your account data plaintext in some old stupid shortcut. Login Wars 2 encrypts the data for you.

Frequently Asked Questions

Is this software safe to use?

LoginWars2 does not include any hacks or cheats and makes use of APIs built into the game by ArenaNet. This includes the Guild Wars 2 Web API and the Command Line Interface.

After I tried logging into my account my screen is black and nothing happens

For some reason Guild Wars 2 won't tell you if your credentials are wrong or your system is not authenticated while using the Command Line Interface. You can fix this problem by logging into the game with the original launcher once.

Where is Login Wars 1?

Originally I wanted to call the app just "Login Wars" but I've decided to add a "2" to not let people think this works with the original Guild Wars.

Is there an OS X version?

No. The app has some initial support for OS X, but as long as the Command Line Interface don't work I'll not support OS X.

I'm having an issue with the overlay, how can I report it?

You can report Issues via the Github Issue Tracker.

Where can I get the source code?

The project is hosted on Github so feel free to fork it.
